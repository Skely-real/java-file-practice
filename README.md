# File Handling Practice (Java)

A small practice project to understand basic file handling and data manipulation in java.

## Concepts Practiced

- File read write using txt files
- ArrayList
- Adding, Updating data 
- Loops and conditions  

## How to run

- Open the project in IntelliJ
- Run FilePractice.java  
- Follow console prompts  

## Notes 

This is a learning project focused on clarity rather than optimization
